# モグラ叩き 仕様書

## メタ情報（registry 登録用）
- id: whack-a-mole
- title: モグラ叩き
- description: 30秒間で穴から顔を出すモグラをクリックして叩く反射神経ゲーム。
- icon: 🔨

## 概要

3x3 に並んだ穴からランダムにモグラが顔を出すので、制限時間 30 秒以内にクリックで叩いてスコアを稼ぐ 1 人用ゲーム。
「モグラ叩き」というお題を、時間経過でモグラの出現テンポが速くなるタイムアタック型のスコアゲームとして解釈した。
対戦要素・永続的なハイスコア保存は持たず、1 プレイ完結（リロードで状態が消えてよい）とする。

## ルール

- フィールドは 3x3 の 9 穴。各穴は「空」または「モグラが出ている」のいずれかの状態を持つ。
- ゲーム時間は 30 秒。0 秒になった時点で終了し、最終スコアを表示する。
- モグラの出現:
  - 一定間隔（スポーン間隔）ごとに、モグラが出ていない穴の中からランダムに 1 穴選び、モグラを出す。
    空いている穴がない場合はその回の出現をスキップする。
  - スポーン間隔は残り時間に応じて段階的に短くなる:
    - 残り 30〜21 秒: 1000ms
    - 残り 20〜11 秒: 800ms
    - 残り 10〜0 秒: 600ms
  - 出たモグラは、叩かれなければ出現から 900ms 後に自動で引っ込む（見逃し。ペナルティなし）。
- スコア:
  - 出ているモグラをクリックで叩く: +1 点。叩かれたモグラは即座に引っ込む。
  - 空の穴をクリック（お手つき）: −1 点。ただしスコアは 0 未満にはならない（下限 0）。
- 同時に複数のモグラが出ていてよい（上限は穴の数 = 9）。
- 勝敗の概念はなく、終了時のスコアがそのまま結果となる。乱数は穴選択のみに使用する
  （`Math.random()` で空き穴インデックスから等確率選択）。

## 操作方法

| 入力 | 動作 |
| --- | --- |
| 「スタート」ボタンをクリック | ゲーム開始（待機画面・終了画面のいずれからも） |
| モグラが出ている穴をクリック/タップ | 叩く（+1 点、モグラが引っ込む） |
| 空の穴をクリック/タップ | お手つき（−1 点、下限 0） |

- キーボード操作は必須ではない（ボタンは button 要素なので Enter/Space での押下は標準挙動に任せる）。

## 画面構成

描画方式は **DOM**（Tailwind CSS でスタイリング）。Canvas は使わない。

共通レイアウト（縦並び・中央寄せ）:

1. ヘッダ行: 「スコア: N」「残り時間: N 秒」を横並びで常時表示。
   - 待機中はスコア 0・残り時間 30 を表示。
2. フィールド: 3x3 グリッド。各セルは丸みのある茶系の「穴」。
   - モグラが出ている穴には絵文字「🐹」（または「🦫」等のモグラ相当 1 文字）を大きく表示。
   - 空の穴は暗い円のみ。
   - 叩いた瞬間はセルに短いフィードバック（例: 背景を一瞬明るくする、絵文字を「💥」に 150ms 差し替える等）を出す。
     実装簡略化のため「叩かれたら即座に消える」だけでも受け入れ基準上は可。
3. フッタ行: 状態に応じたボタン・メッセージ。

状態別:

- **待機（プレイ前）**: フィールドは全穴空で表示。中央またはフッタに「スタート」ボタンと簡単な説明文
  （「モグラをクリックして叩こう！ 空振りは −1 点」）を表示。
- **プレイ中**: フィールドが有効。スタートボタンは非表示（またはリスタート用に「やめる」は置かない。プレイ中の中断機能は不要）。
- **終了**: フィールド上またはフッタに「終了！ スコア: N」と「もう一度」ボタンを表示。
  フィールドのモグラはすべて引っ込め、クリックしても反応しない。

## 状態遷移

```
待機 (idle)
  └─ スタートボタン → プレイ中 (playing)
        ・タイマー 30 秒カウントダウン開始（1 秒刻みで表示更新）
        ・スポーンタイマー開始
プレイ中 (playing)
  └─ 残り時間 0 → 終了 (finished)
        ・全タイマー停止、全モグラを引っ込める
終了 (finished)
  └─ 「もう一度」ボタン → プレイ中 (playing)
        ・スコア 0、残り時間 30、全穴空にリセットして即開始
```

- リスタートは終了画面の「もう一度」ボタンで行う。待機画面に戻る遷移は不要。
- コンポーネントのアンマウント時にはすべてのタイマー（interval / timeout）を解除する。

## エッジケース

- **同一モグラへの連打**: 1 匹のモグラで加点されるのは 1 回のみ。叩いた瞬間に穴を空状態にするため、
  2 回目以降のクリックは「空の穴クリック」= お手つき（−1）として扱う。
- **スコア下限**: お手つきでスコアが負になる場合は 0 に留める（例: 0 点でお手つき → 0 点のまま）。
- **引っ込む直前のクリック**: クリック時点で穴にモグラがいれば加点。自動引っ込みタイマーとの競合は
  「state 上モグラがいるかどうか」で判定し、加点した場合は該当穴の自動引っ込みタイマーを無効化する
  （無効化しない実装の場合も、既に空の穴を空にするだけで副作用がないこと）。
- **空き穴なしでのスポーン**: 9 穴すべてにモグラがいる場合、その回の出現はスキップ（エラーにしない）。
- **終了間際の出現**: 残り時間 0 になった時点で出ているモグラはすべて即座に引っ込め、以降のクリックは無効。
- **終了後のクリック**: finished 状態ではフィールドのクリックはスコアに一切影響しない。
- **スタート連打**: playing 中はスタート/もう一度ボタンを表示しない（または disabled）ため、二重開始は起きない。
- **タイマー精度**: 残り時間表示は整数秒でよい。多少の誤差（±100ms 程度）は許容する。

## 受け入れ基準

1. ゲーム一覧に「モグラ叩き」（アイコン 🔨）が表示され、選択するとゲーム画面が開く。
2. 初期表示は待機状態で、スコア 0・残り時間 30 秒・空の 3x3 の穴と「スタート」ボタンが表示される。
3. 「スタート」をクリックするとカウントダウンが始まり、モグラが穴からランダムに出現し始める。
4. 出ているモグラをクリックするとスコアが 1 増え、そのモグラが即座に引っ込む。
5. 叩かずに放置したモグラは約 1 秒以内に自動で引っ込む（スコアは変化しない）。
6. 空の穴をクリックするとスコアが 1 減る。ただしスコアが 0 のときは 0 のまま変化しない。
7. 残り時間が減るにつれてモグラの出現ペースが速くなる（序盤より終盤の方が明らかに多く出る）。
8. 残り時間が 0 になると「終了！ スコア: N」と最終スコアが表示され、穴をクリックしてもスコアが変化しない。
9. 終了画面の「もう一度」をクリックすると、スコア 0・残り時間 30 秒にリセットされて即座に再プレイが始まる。
